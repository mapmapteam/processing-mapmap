Processing - MapMap example
===========================
Example of rendering Processing to MapMap using JPEG images and OSC.

Author: Alexandre Quessy
License: public domain
Date: October 2015
Syntax of this file: reStructuredText


Installation
------------
Uncompress and copy the oscP5 library to your sketches folder.
Run MapMap with the project::

  mapmap mapmap_processing.mmp

Run the processing sketch.
